Paste your Firebase Admin service account JSON into a new file here named:

  firebase-service-account.json

This file is .gitignored (do not commit it). Then run:

PowerShell:
  setx GOOGLE_APPLICATION_CREDENTIALS "C:\Users\llaks\OneDrive\Desktop\train-fluence\server\firebase-service-account.json"
  setx FIREBASE_PROJECT_ID "train-25c2c"
  (Close and reopen PowerShell)
  cd C:\Users\llaks\OneDrive\Desktop\train-fluence\server
  npm run dev

Alternatively, create server/.env with:
  GOOGLE_APPLICATION_CREDENTIALS=C:\Users\llaks\OneDrive\Desktop\train-fluence\server\firebase-service-account.json
  FIREBASE_PROJECT_ID=train-25c2c

and run:
  cd C:\Users\llaks\OneDrive\Desktop\train-fluence\server
  npm run dev
